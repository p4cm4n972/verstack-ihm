📦 Verstack.io – Notes de Version

✅ Version 1.0.0 – Lancement Officiel

Date : Mai 2025

Cette première version marque le lancement public de Verstack.io, votre référentiel interactif pour :

  • 🌐 Consulter les versions courantes, LTS et standards des langages et frameworks.
  • 📊 Explorer les tendances technologiques par domaine : web, mobile, IA, devops, game, etc.
  • 🛠️ Visualiser les taux de support de chaque version via des barres de progression dynamiques.
  • 📰 Lire des articles et éditos sur l’évolution des technologies.
  • 🛍️ Accéder à la boutique Red Squiggly pour les développeurs geeks 👕.
  • 💼 Créer un profil personnalisé avec vos stacks favoris.
  • 📈 Suivre des statistiques de popularité des technos (Stack Overflow, GitHub, TIOBE...)

👉 Cette version pose les fondations d’un portail pensé pour les développeurs, les formateurs, les recruteurs et les curieux du code.

🚀 Ce qui arrive en Version 2.0

Prévue : Q3–Q4 2025

La prochaine version majeure apportera une connexion directe à vos projets GitHub / GitLab :

  • 🔗 Liaison de vos repositories à votre profil Verstack.io
  • 📦 Détection automatique des technos utilisées dans chaque projet
  • 📍 État de santé des stacks : alertes si une technologie de votre projet est obsolète ou en fin de support
  • 📅 Suivi des mises à jour à prévoir dans vos stacks (Angular, Node.js, Python, etc.)
  • 🧠 Suggestions intelligentes pour anticiper les migrations et changements majeurs

⚙️ Cette fonctionnalité fera de Verstack.io un assistant de veille technique connecté à vos besoins réels.
