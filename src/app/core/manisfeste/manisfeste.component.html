Le Manifeste

Nous ne sommes pas contre la technologie. Nous sommes contre la passivité.

Le mobile est devenu l’extension de la main.
On le scrolle. On le swipe. On le subit.

Mais le geek, lui, construit.
Il tape. Il déploie. Il explore.
Nous ne scrollons pas. Nous créons.

Le mobile vous donne du contenu.
Le terminal vous donne le contrôle.
Sur mobile, vous regardez.
Sur clavier, vous devenez acteur.

Mobile is not Geek.

Un geek, ce n’est pas juste quelqu’un derrière un écran.
C’est quelqu’un qui tire parti de la technologie,
pas quelqu’un qui s’y perd.

Ici, on préfère un IDE à une app.
Un clavier mécanique à un écran tactile.
Un git push à une notif.

Nous ne sommes pas anti-tech.
Nous sommes pro-esprit critique.
Pro-construction. Pro-maîtrise.

Le mobile est un outil.

Le clavier est une arme.

À chacun son camp.

#MobileIsNotGeek
